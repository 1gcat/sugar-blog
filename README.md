## 运行本项目
博客基于[vitepress](https://vitepress.vuejs.org/)构建

先安装 `pnpm`

```sh
npm i -g pnpm@7
```

```sh
# 安装依赖
pnpm install

# 开发启动
pnpm dev
# or
npm run dev

# 构建
pnpm build
# or
npm run build
```
